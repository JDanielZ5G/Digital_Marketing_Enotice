# E-Notice Board 📋

**The Source of Truth** — A real-time digital notice platform for Uganda Christian University's Department of Computing and Technology.

## Overview

E-Notice Board is a mobile-first web application designed to eliminate the chaos of physical notice boards on campus. Students log in with their registration number to receive instant, offline-cached alerts for results, timetables, and critical memos — no paper, no delays, no excuses.

## Features

- **📱 Instant Mobile Access** — Receive every notice the moment it's posted, with offline caching for access anywhere on or off campus.
- **⚡ Zero Missed Deadlines** — Priority badges highlight urgent memos, exam timetables, and registration alerts delivered in real time.
- **🔒 Secure & Reliable** — Role-based access with verified, time-stamped notices that are automatically archived for reference.
- **🌐 24/7 Availability** — Always-on platform serving 6+ academic programs with 100% paperless delivery.

## Tech Stack

- **HTML5** — Semantic markup with SEO best practices
- **CSS3** — Custom design system with CSS variables, responsive layouts, and smooth animations
- **JavaScript** — Scroll-triggered animations with Intersection Observer, mobile navigation
